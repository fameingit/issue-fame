# Fame In Git • issue-fame
Get the Quickdraw achivment in Github.

## Here's how this works

Github gives the **Quickdraw** achivement when you create a issue and close it within 5 minutes of opening.

## What you do

Head to [New Issue](https://github.com/fameingit/issue-fame/issues/new?template=gimme_quickdraw.md), create a new one and follow the instructions in there.

Done! Enjoy the Quickdraw achivment.
